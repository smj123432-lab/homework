# 세 번째 회고

## 멋쟁이 사자처럼 프론트엔드 16기 세 번째 회고

#### 성장했다고 느끼는 부분

그새 3주 차가 되어 css 스타일링과, ui, 레이아웃을 만드는 법을 배우고 직접 만들어보기도 했습니다.

확실히 성장했다고 느낀 것은 혼자선 아예 스타일링을 건들 생각도 하지 못했고 어떻게 구조를 짜야할 지도 몰랐습니다.

근데 차츰차츰 사진을 보며 따라할 수 있게 되었고 뭔가를 구성할 때 아 이걸 쓰면 되겠구나 생각나는 것들이 생겨났습니다. 저번에 성장했다고 느낀 **논리적으로 구조화**뿐 아니라 **추론하는 능력**도 향상됨을 느꼈습니다.

- **레이아웃의 활용**: 부모 요소에 어떤 속성을 주어야 하위 요소들이 정렬되는지, 어떤 식으로 꾸며지는 지를 조금이나마 알게되었습니다.
- **스타일링의 접근성**: 계속해서 접근성에 대해 고민하게 되었습니다. 또한, 접근성에 대한 고민은 끊임없이 해야한다는걸 다시 한 번 알게됐습니다.
- **반응형 스타일링**: 그저 보기 좋게 스타일링을 하는 것 뿐이 아닌 이 요소를 클릭했을때 생기는 반응을 짜는 방법을 배웠습니다.

#### 어려웠지만 해결하는 과정

- **ui 구성**: 전혀 감도 잡히지 않아 고통스러웠지만 될떄까지 붙잡아보니 조금씩 만들 수 있게 되었습니다.
- **반응형 구조**: 아직 어렵고 고통스럽지만 이 또한 포기하지 않고 하다보니 조금씩 이해할 수 있게 되었습니다.

#### 3주차에 들어가며 하는 다짐

- **포기하지 않겠습니다**: 전혀 감도 잡히지 않는 것들을 볼때마다 포기에 대한 욕구가 솓구치지만 포기하지 않겠습니다.
- **건강 관리**: 저번주와 마찬가지로 건강 관리 잘하겠습니다.
